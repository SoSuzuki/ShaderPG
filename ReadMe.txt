～各ブランチについて～
○master
　・鏡面反射光を適用した最初の段階。物体貫通して光が届いており、影が上手く反映できてない
　・光源位置も固定
○specular
　・masterにある完成品を作業してたブランチ
○lightSource
　・影と輝度(shininess)も再現した最新のブランチ。12/15時点の更新は実行可能な状態
　・光源位置も操作可能

～操作～
○master
　・WASDで視点移動
　・矢印キーで視点角度調整
　・SPACEとLightCTRLで昇降
　・Rで視点位置リセット

○lightSource
　・WASDは同じ
　・QとEで視点角度調整
　・矢印キーは光源位置
　・SPACEとLightCTRLで光源を昇降
　・Rも同じ
