～各ブランチについて～
○master
　・鏡面反射光を適用した最初の段階。物体貫通して光が届いており、影が上手く反映できてない
　・光源位置も固定
○specular
　・masterにある完成品を作業してたブランチ
○lightSource
　・影と輝度(shininess)の概念を入れたブランチ。12/18時点の更新は実行可能な状態
　・光源位置も操作可能
○Postarization
　・ポスタリゼーションを試しているブランチ
～操作～
○master
　・WASDで視点移動
　・矢印キーで視点角度調整
　・SPACEとLightCTRLで昇降
　・Rで視点位置リセット

○lightSource
　・WASDはmasterと同じ
　・QとEで視点角度調整
　・矢印キーは光源位置の操作
　・SPACEとLightCTRLで光源を昇降
　・Rもmasterと同じ


◎作っていてわかりづらい、難しかったところ
　・計算式をプログラムに書き出す過程で、コンスタントバッファに入れるべき情報をどこまで
　増やしていいのかよく分からず、使用する箇所で分割するという発想が無かったため、
　情報が足りなかったり正確な値を取れず、変な色になってしまった。

◎作っていてわかったこと、学んだこと
　・今まではバッファに情報を渡した結果どうしてそうなるのか全く理解できていなかった。
　しかし、今回の課題でコンスタントバッファに各種情報(色や座標)を渡す必要性と
　それらをどうやって頂点シェーダとピクセルシェーダで使っていくかを多少は分かってきたと思う
◎作ってみた感想
　・シェーダーについては全く理解してない状態でスタートしたためよく分からないことが多かったが、
　試行錯誤と先生の授業を聞いていく内に少しずつ分かってきて面白くなってきた。今回は
　面光源だったが、点光源もできるとさらに理解が深まり表現の幅が広がりそうなので
　暇なときに試してみたいと考えている。
